# GCTM
This is an implementation of Graph Convolution Topic Model for Data Streams

Data descriptions:
* Training file, we used the bag of words format.
* Testing folder (E.g data).
* Graph, we saved by format: node1 \tab node2 \tab weight.
Requirements:
* python 3.7.
* pytorch 1.2.0
Run the demo:
* python runGCTM.py
